# CartPicks

CartPicks 是一个创业方向 demo，目标是验证一个围绕 Amazon affiliate 模式的内容导购站长什么样。

当前第一版方向：

- 垂类定位：送礼 + 家居好物
- 页面形态：首页 + 榜单区 + 单品卡 + 选品框架
- 商业路径：用户浏览内容 -> 点击跳转 Amazon -> 完成购买 -> 获得 affiliate 佣金

当前项目是静态前端 demo，按钮使用 Amazon 搜索链接占位。后续真实上线时需要：

1. 接入 Amazon Associates
2. 替换成真实 affiliate 链接
3. 增加 affiliate disclosure 页面
4. 补充更多榜单与商品内容
